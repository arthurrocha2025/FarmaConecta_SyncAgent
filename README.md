# FarmaConecta_SyncAgent
Aplicativo Windows, projetado para rodar em segundo plano (Service) nos servidores locais das farmácias. Sua responsabilidade é ler bancos de dados heterogêneos (Firebird, SQL Server, PostgreSQL) utilizando "Receitas" (Templates JSON) dinâmicas baixadas da nuvem, mapear os dados para um formato padrão, criptografá-los e transmiti-los via API.
